# 🎨 frontend/styles/

This folder contains the **CSS (Cascading Style Sheets)** that control how the webpage looks.

---

## 🗂️ Files in this Folder

| File | What it does |
|---|---|
| `main.css` | The **main stylesheet** for the entire application. It sets colors, fonts, spacing, layout, button styles, animations, responsive design rules, and more. |

---

## 💡 What is CSS?

CSS is like the **interior design** of a website. HTML creates the structure (walls, rooms), and CSS makes it look beautiful (paint, furniture, lighting).

For example, CSS in `main.css` controls things like:
- The dark background color of the page
- The size and color of buttons
- How graphs are arranged in a grid
- The smooth animations when switching tabs
- How the layout adjusts on smaller screens (responsive design)

---

## ✏️ Want to Change the Look?

If you want to change colors, fonts, spacing, or layout, this is the file to edit. Look for CSS variables at the top of the file — these are like "settings" that control the whole theme.
